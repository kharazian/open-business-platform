# V3 Task 001: Advanced Permissions

## Goal

Finish organization-aware access control for users, roles, groups, departments, records, reports, actions, and fields.

## Acceptance Criteria

- [x] Groups can be managed and assigned to users.
- [x] Departments can be managed with manager users.
- [x] Form permissions support record access scopes.
- [x] Record list/detail/edit/delete use backend scoped access.
- [x] Records can be assigned to users or groups with scoped `assign` access.
- [x] Record status changes require scoped `change_status` access.
- [x] Reports support explicit view/export/manage permissions.
- [x] Report run/export rows are filtered through scoped record access.
- [x] Hidden fields are omitted from record/report/chart/dashboard responses.
- [x] Read-only fields cannot be changed through record edit.
- [x] Users & Access exposes users, roles, departments, groups, scoped form access, report access, and field rules.
- [x] Documentation and verification are updated.

## Current Status

Complete for the current V3 slice.
